# Unsplash Search App

## Установка

```
npm install
```

Создайте файл `.env` и добавьте ключ:

```
VITE_UNSPLASH_ACCESS_KEY=ВАШ_КЛЮЧ
```

## Скрипты

- `npm run dev` – режим разработки
- `npm run build` – production сборка
- `npm run preview` – превью production билда
- `npm run test` – unit-тесты (jest + RTL)

## Запуск

```
npm run dev
```
